# ZOONA - متجر زونا الإلكتروني السوداني

**ZOONA** هو تطبيق متجر إلكتروني متكامل (PWA) مصمم خصيصاً للسوق السوداني، يوفر تجربة تسوق سلسة وسريعة لشراء الموبايلات، الأجهزة الإلكترونية، والمنتجات المنزلية.

## 🚀 الميزات الرئيسية

*   **تطبيق ويب تقدمي (PWA):** يمكن تثبيت المتجر كأيقونة على الشاشة الرئيسية للهاتف ليعمل كتطبيق أصلي، مع دعم العمل في وضع عدم الاتصال (Offline).
*   **تكامل مع Supabase:** قاعدة بيانات سحابية لإدارة المنتجات والطلبات بشكل فوري وآمن.
*   **إشعارات Firebase (FCM):** وصول تنبيهات فورية للمستخدمين حول العروض الجديدة وتحديثات الطلبات.
*   **نظام سلة تسوق متكامل:** إضافة وتعديل المنتجات في السلة بسهولة مع حفظ البيانات محلياً.
*   **تتبع الطلبات:** قسم خاص لمتابعة حالة الطلبات السابقة.
*   **بحث وتصفية ذكي:** البحث عن المنتجات بالاسم أو القسم وتصفية المنتجات حسب الفئات (موبايلات، إلكترونيات، إلخ).
*   **روابط عميقة (Deep Linking):** دعم التنقل المباشر للمنتجات عبر روابط صديقة لمحركات البحث (SEO-friendly slugs).
*   **واجهة مستخدم متجاوبة (RTL):** تصميم عصري وسريع يدعم اللغة العربية بشكل كامل (من اليمين لليسار).

## 🛠️ التقنيات المستخدمة

*   **Frontend:** Vanilla JavaScript, HTML5, CSS3 (مع خطوط Google Fonts وأيقونات Font Awesome).
*   **Backend (Local Dev):** Node.js server لتقديم الملفات الساكنة وإدارة API بسيطة للمنتجات.
*   **Database:** Supabase (PostgreSQL) لتخزين بيانات المنتجات.
*   **Notifications:** Firebase Cloud Messaging (FCM).
*   **Deployment:** Vercel مع إعدادات SPA Routing.

## 📁 هيكل المشروع

*   `index.html`: الملف الرئيسي الذي يحتوي على واجهة المستخدم ومنطق التطبيق (JavaScript/CSS).
*   `server.js`: خادم Node.js محلي للتطوير والاختبار.
*   `manifest.json`: إعدادات تطبيق الويب التقدمي (PWA).
*   `service-worker.js`: إدارة التخزين المؤقت والإشعارات لضمان عمل التطبيق بدون إنترنت.
*   `vercel.json`: تكوين توجيه الروابط (Routing) لخدمة Vercel.
*   `assets/`: الصور والأصول الخاصة بالتطبيق.
*   `products.json`: نسخة احتياطية محلية لبيانات المنتجات.

## 💻 التشغيل المحلي

1.  تأكد من تثبيت [Node.js](https://nodejs.org/).
2.  قم بتنزيل المشروع.
3.  شغل الخادم باستخدام الأمر:
    ```bash
    node server.js
    ```
4.  افتح المتصفح على الرابط: `http://localhost:5000`
